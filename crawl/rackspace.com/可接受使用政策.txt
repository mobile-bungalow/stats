<p> 本 AUP 中所用且未在此另作定义的大写术语，应具备相关协议中所赋予的含义。</p>
<p> 有关本政策的查询将引至 abuse@rackspace.com。</p>
<p>滥用</p>

<ul>
<ul>
<li>可能违反任何适用法律、法规或条例的任何活动或行为，包括数据保护和隐私法律以及涉及未经请求发送商业电子消息的法律； </li>
<li>在未经所有者授权的情况下使用互联网账户或计算机；</li>
<li>未经授权访问或使用数据、系统或网络，包括在未经系统或网络所有者明确授权的情况下，任何旨在探查、扫描或测试系统或网络弱点，或破坏安全或认证措施的尝试举动；</li>
<li>在未经系统或网络所有者明确授权的情况下，监测任何网络或系统上的数据或流量。</li>
<li>蓄意、故意或肆意将任何病毒或其他传染性代码引入相关服务；</li>
<li>通过欺骗性手段（例如网络钓鱼、互联网欺诈、密码盗取、爬网和收割）收集或使用信息，包括电子邮件地址、屏幕名称或其他标识符；</li>
<li>在电子邮件或新闻组发帖中使用任何虚假的、误导性或欺骗性 TCP-IP 数据包标题信息； </li>
<li>分发秘密收集或传送用户相关信息的软件；</li>
<li>分发广告投送软件，但以下情况除外： (i) 用户在获得有关软件性质的清楚易懂的通知的情况下，明确同意下载和安装该软件；以及 (ii) 软件可使用主要操作系统中为增删软件而装设的标准工具（如 Microsoft 的“添加/删除”工具）轻松移除；</li>

<li>任何有意隐瞒或掩盖身份或联系信息的活动，包括省略、删除、伪造或谎报任何传送或识别信息，例如回信和 IP 地址； </li>
<li>干扰向 Rackspace 或其他网站任何用户提供的服务，包括但不限于邮件轰炸、邮件泛滥、蓄意系统过载尝试以及广播攻击； </li>
<li>任何直接或间接导致我们的任何 IP 空间被列入任何反滥用数据库（例如 Spamhaus）的行动；</li>
<li>实施任何违反您站点托管或接入所在任何司法辖区的法律法规所规定的必需许可证、行为守则或必要技术标准的赌博行为；或者</li>
<li>任何以其他方式违法的行动，或教唆作出在适用于您或 Rackspace 的法律下属于非法的行动</li>
</ul>
</ul>

<p>您不得在相关服务上或通过相关服务发布、传送或保存 Rackspace 合理认为存在下列情况的任何内容或任何内容链接：</p>

<li>依据任何司法辖区的消费者保护法均判定为不公平或欺诈性，包括连锁信和传销;</li>

<p>凡属相关服务一旦失效或故障可能导致任何人员死亡或严重身体伤害，或造成物理或环境破坏的情况，一律不得使用相关服务。 例如，您不得在结合使用航空器或其他形式大规模载人交通工具、或核设施或化学设施的情况下使用或者允许任何他人使用相关服务。  </p>

<p>您必须遵守您所在司法辖区内群发或商业邮件的适用法律和法规。 此外，您的群发或商业电子邮件必须符合下列要求：</p>
<ul>
<ul>

<li>您必须在与该电子邮件相关的任何网站的显著位置上发布投诉专用电子邮件地址（例如 abuse@yourdomain.com），而且您必须迅速回复已发送至该地址的邮件；</li>
<li>您预定的收件人已通过某些确认方式（例如选择性加入程序）同意接收电子邮件，在接到收件人或 Rackspace 请求后的 72 小时内，您可以就上述同意生成证据。 </li>
<li>您必须使用合理的方法，确保给出同意的人正是给予同意之对象电子邮件地址的所有者。 </li>
<li>您必须在邮件消息正文中或电子邮件的“收件人”行中输入收件人的电子邮件地址。  </li>
<li>您必须尊重撤销同意的决定，并将相关信息通知给收件人；</li>
</ul>
</ul>
